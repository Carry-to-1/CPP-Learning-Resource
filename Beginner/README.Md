# Course Syllabus: ITEX 103 Object Oriented Programming 

## Course Description and Goals
The course covers the concepts of Object-Oriented Programming using the C++ programming language. The main objective of this course is to deliver and to make learners understand the basic concepts of OOP, realize the peculiarity of the Object-Oriented Paradigm, and implement these concepts in real-world projects. It covers OOP concepts like objects, classes, polymorphism, overloading, inheritance, exceptions, file handling, and more.

## Learning Outcomes
On completion of this course, the students will be able to:
- Understand the specialty and importance of the Object-Oriented Approach.
- Implement the programming features of OOP.
- Create programs and projects using the C++ programming language.
- Learn any other OOP language based on the basics delivered through this course.

## Content with Specific Objectives

### Unit One: Introduction to OOP (5 hrs.)
**Specific objectives:**
- Compare procedure and object-oriented programming.
- Describe the features of object-oriented programming.
- List different types of OOPs.
- Compare coding structures of C and C++.
- Understand the details of C++ program structure.
- Revise the concepts of control structures in C++.

**Contents:**
- Software Crisis and emergence of OOP concept
- Procedural vs OOP Languages
- Features of OOPs
- Programming structures in C++
- Basic programming terminologies: tokens, data types, operators, etc.
- Programming Construct Control Structures

### Unit Two: Object and Classes (8 hrs.)
**Specific objectives:**
- Understand the basics of classes and objects.
- Be able to initialize objects in accordance with their class.
- Implement the use of the `this` operator.
- Understand the idea of access specifiers.
- Understand and implement the use of constructors and destructors in C++.

**Contents:**
- Introduction to Objects and Classes
- Defining a class with data members and member functions
- Access specifiers
- Initializing objects
- Static Data Members
- Static Member Functions
- `this` operator
- Constructor
- Destructor

### Unit Three: Operator Overloading (7 hrs.)
**Specific objectives:**
- Understand and demonstrate the use of operator overloading in C++.
- Implement unary operator overloading.
- Implement binary operator overloading.
- Implement the concept of operator overloading in member functions and non-member functions.

**Contents:**
- Introduction to operator overloading
- Overloadable operators
- Syntax and rules of operator overloading
- Unary operator overloading
- Binary Operator overloading
- Operator overloading with Member and non-member functions
- Overloading binary operator using a friend function

### Unit Four: Inheritance (7 hrs.)
**Specific objectives:**
- Understand and implement the concept of inheritance.
- Understand base class and derived classes.
- Implement constructors and destructors for derived class in C++.
- Understand the concept of abstract class and virtual base class.
- Implement use of access specifiers in C++ programs.

**Contents:**
- Introduction to inheritance
- Base class and derived class
- Derived class constructor
- Derived class destructor
- Single inheritance and multiple inheritance
- Virtual Base Classes and Abstract Classes
- Public, Private, and Protected access and control

### Unit Five: Polymorphism (9 hrs.)
**Specific objectives:**
- Understand the concept of polymorphism.
- Implement `new` and `delete` operators.
- Understand and implement virtual functions in C++ programs.
- Understand the concept of dynamic binding.

**Contents:**
- Introduction to polymorphism
- `new` and `delete` operator
- Pointer to objects
- Pointer to derived class
- Virtual Functions
- Friend Functions
- `dynamic_cast` operator
- `typeid` operator

### Unit Six: Templates (4 hrs.)
**Specific objectives:**
- Understand the idea of templates and frameworks.
- Implement class templates in C++.
- Understand function templates.

**Contents:**
- Concept of templates
- Class templates
- Function Templates
- Concept of Frameworks

### Unit Seven: Exception Handling (5 hrs.)
**Specific objectives:**
- Understand the idea of error and exceptions.
- Implement ideas to handle exceptions in C++.
- Implement rethrowing an exception in C++.
- Handle various types of exceptions in C++.

**Contents:**
- Error Handling
- Exception handling constructs (`try`, `catch`, `throw`)
- Multiple exception handling
- Catching all exceptions
- Exception with arguments
- Re-throwing an exception
- Handling uncaught and unexpected exceptions
